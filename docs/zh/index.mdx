IronClaw 是一个安全、开源的 AI 智能体框架，基于 Rust 构建，部署在 NEAR AI Cloud 上。它可以创建能够访问您工具和服务的 AI 智能体，同时确保您的凭证安全和隐私。

<Card title="快速开始" icon="rocket" href="/quickstart" horizontal>
  几分钟内部署您的第一个智能体。
</Card>

---

## 核心能力

<CardGroup cols={3}>
  <Card title="多渠道" icon="message" href="/channels">
    通过浏览器、Telegram、终端界面或 HTTP Webhook 访问 IronClaw。
  </Card>

  <Card title="安全优先" icon="shield-check" href="/security">
    多层防护体系：安全层、WASM 沙箱、Docker 隔离与加密密钥。
  </Card>

  <Card title="多提供商 LLM" icon="brain">
    可从 7 种以上提供商中选择，包括 NEAR AI、Anthropic、OpenAI、Ollama、Tinfoil 等。
  </Card>

  <Card title="技能系统" icon="puzzle" href="/capabilities/skills">
    通过 ClawHub 注册表中的 SKILL.md 提示扩展增强能力。
  </Card>

  <Card title="并行任务" icon="briefcase" href="/capabilities/jobs/jobs">
    通过状态机与自修复机制并发执行多个任务。
  </Card>

  <Card title="持久记忆" icon="database" href="/capabilities/memory/identity">
    结合身份文件与 heartbeat 系统，提供混合搜索（全文检索 + 向量检索）。
  </Card>
</CardGroup>
