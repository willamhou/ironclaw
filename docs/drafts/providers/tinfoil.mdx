Tinfoil provides hardware-attested Trusted Execution Environment (TEE) inference вҖ” neither Tinfoil nor the cloud provider can see your prompts.

## Overview

Tinfoil runs models inside hardware-attested TEEs:

- **Hardware-attested** вҖ” Intel TDX or AMD SEV-SNP
- **End-to-end encrypted** вҖ” Only you can decrypt responses
- **Verifiable** вҖ” Cryptographic proof of execution
- **Zero-knowledge** вҖ” Provider sees only encrypted data

## Configuration

```bash
# ~/.ironclaw/.env

LLM_BACKEND=tinfoil
TINFOIL_API_KEY=your-api-key
TINFOIL_MODEL=kimi-k2-5
```

## Getting an API Key

1. Visit https://tinfoil.sh
2. Create an account
3. Generate an API key
4. Copy the key

## Available Models

| Model | Description |
|-------|-------------|
| `kimi-k2-5` | Moonshot AI Kimi K2.5 (default) |

## Setup

### Via Wizard

```bash
ironclaw onboard
```

1. Step 3: Select "Tinfoil"
2. Enter your API key
3. Step 4: Confirm model

### Manual Configuration

```bash
export LLM_BACKEND=tinfoil
export TINFOIL_API_KEY=your-api-key
export TINFOIL_MODEL=kimi-k2-5
```

Restart IronClaw:
```bash
ironclaw run
```

## Privacy Guarantees

- **Tinfoil** вҖ” Cannot see prompts (attested code)
- **Cloud provider** вҖ” Only sees encrypted traffic
- **IronClaw** вҖ” Your local client, you control
